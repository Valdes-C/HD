📊 Repozytorium Przedmiotu: Hurtownie Danych (DWH)
Witajcie na zajęciach z Hurtowni Danych!

To repozytorium zawiera komplet materiałów, skryptów oraz danych potrzebnych do realizacji ćwiczeń i wykładów. Naszym głównym celem jest przejście od surowych, "brudnych" danych z plików tekstowych do profesjonalnej architektury analitycznej, którą na koniec podepniemy pod aplikację w Spring Boot.

🛠️ Wymagania Techniczne (Stack)
Aby w pełni uczestniczyć w zajęciach i wykonywać zadania praktyczne, upewnijcie się, że macie zainstalowane następujące oprogramowanie:

Baza danych: Microsoft SQL Server (wersja Developer lub Express)

Zarządzanie bazą: SQL Server Management Studio (SSMS)

ETL: SQL Server Data Tools (SSDT) z rozszerzeniem do SSIS

Backend / Dashboard: Java (JDK 17+) oraz środowisko do Spring Boot (np. IntelliJ IDEA)

📦 Zbiór Danych do Ćwiczeń (Dataset)
Na zajęciach pracujemy na fikcyjnym zestawie danych reprezentującym sprzedaż leków z dużej sieci aptek.

Plik źródłowy: dane/hurtownia_lekow.csv

Rozmiar: 1000 wygenerowanych rekordów (transakcji).

Opis: Dane zawierają informacje o producencie, cenie, dacie sprzedaży, mieście oraz statusie refundacji. Będziemy je sukcesywnie "czyścić" i ładować do naszej docelowej bazy.
